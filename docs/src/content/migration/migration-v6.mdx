# Migrating from v5 to v6

### Update version

```bash
npm install @dataxpdtn/mui-json-viewer@^6.0.0
npm install @mui/material@^9.0.0
```

For `MUI` users, you should also check the [MUI v9 migration guide](https://mui.com/material-ui/migration/upgrade-to-v9/).

### Breaking changes

This package supports MUI v9. Hence, you need to upgrade to v9 first, before you upgrade this package. No other changes should be required.
